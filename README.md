# Snapgram

Snapgram is a social media web app.

## Installation

```bash
$ git clone
$ cd snap-gram
$ npm install
$ npm start
```

## Features

- Create, edit and delete posts
- Create, edit and delete comments on posts
- Like/dislike post
- Add/remove post to/from bookmark
- Follow/unfollow user
- User profile
- Edit User profile
- Sort posts
- Authentication

## Tech Stack

- ReactJS
- React Router v6
- Redux Toolkit
- Tailwind CSS
